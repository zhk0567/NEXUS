# 每日故事 - 对照组应用

## 📱 项目简介

这是一个专门为老年人设计的Android故事阅读应用，作为NEXUS项目的对照组。应用启动后直接显示当天的故事内容，提供温馨、感人的故事，帮助老年人享受阅读的乐趣。

## 🎯 设计目标

- **对照组功能**：作为NEXUS智能对话系统的对照组，测试纯文字阅读对老年人的影响
- **老年人友好**：大字体、简洁界面、温馨内容
- **每日更新**：每天提供一个新的故事
- **测试功能**：提供日期选择功能，方便测试不同日期的故事

## ✨ 主要功能

### 📖 故事阅读
- **直接展示**：打开应用即可看到当天的故事
- **语音播放**：支持TTS语音朗读，0延迟播放预录制音频
- **温馨内容**：精心挑选适合老年人的故事，包括：
  - 温馨故事：邻里关爱、家庭温暖
  - 家庭故事：孝顺父母、祖孙情深
  - 友谊故事：老友重逢、邻里互助
  - 智慧故事：人生感悟、生活哲理
  - 传统故事：文化传承、传统节日

### 🎨 界面设计
- **与原应用一致**：保持与NEXUS应用相同的UI风格和主题
- **大字体支持**：小、中、大三种字体大小选择
- **主题切换**：支持白天/夜间模式
- **简洁布局**：清晰的标题和内容分区

### 📅 日期选择
- **日历界面**：点击右上角日期按钮打开日历选择器
- **默认日期**：应用启动时默认显示2025年1月1日的故事
- **范围限制**：支持选择2025年1月的所有31天
- **即时切换**：选择日期后立即显示对应故事

### 🔊 语音功能
- **预录制音频**：所有故事音频已提前录制并内置在应用中
- **0延迟播放**：点击播放按钮即可立即播放，无需网络连接
- **播放控制**：播放/停止按钮，简单易用
- **离线使用**：完全离线工作，不依赖网络连接

### ⚙️ 个性化设置
- **主题模式**：白天/夜间模式切换
- **字体大小**：小、中、大三种字体选择
- **设置持久化**：设置自动保存，下次启动时保持

## 🚀 快速开始

### 1. 环境要求
- Android Studio (最新版本)
- Android SDK 24+
- Kotlin 1.9.10+

### 2. 编译运行
```bash
# 编译应用
./gradlew assembleDebug

# 安装到设备
./gradlew installDebug
```

### 3. 使用说明
1. **启动应用**：安装后点击图标启动
2. **阅读故事**：应用启动后直接显示当天故事
3. **语音播放**：点击播放按钮收听故事语音朗读（0延迟）
4. **选择日期**：点击右上角日期按钮选择其他日期
5. **调整设置**：点击左上角设置按钮调整主题和字体

### 核心技术

- **数据管理**：本地故事数据管理
- **主题系统**：完整的深色/浅色主题支持
- **字体系统**：动态字体大小调整
- **语音系统**：预录制音频播放，0延迟响应
- **离线功能**：完全离线工作，无需网络连接

## 📊 对照组设计

### 与NEXUS应用的对比
| 特性 | NEXUS应用 | 故事对照组 |
|------|-----------|------------|
| 交互方式 | 语音对话 | 文字阅读+语音播放 |
| 内容类型 | AI生成对话 | 预设故事 |
| 用户参与 | 主动提问 | 被动阅读+语音控制 |
| 技术复杂度 | 高（ASR+TTS+AI） | 低（UI+预录制音频） |
| 个性化程度 | 高（对话式） | 中（设置调整） |
| 语音功能 | 实时TTS | 预录制音频播放（0延迟） |
| 网络依赖 | 需要网络 | 完全离线 |
